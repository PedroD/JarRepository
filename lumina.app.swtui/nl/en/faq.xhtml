Frequently Asked Questions
 

Frequently asked questions about iCANgraph have their answers here.

  • What is iCANgraph?
    iCANgraph is the easiest way to control an iLight installation.
    There are two applications available, one allows control and command and the other allows configuration and management.
  • What can iCANgraph do?
    iCANgraph can control and manage devices on any iLight installation via manual control or predefined scenarios.
  • What skills are necessary to use iCANgraph?
    iCANgraph can be used by all users, since no particular skills are required.
  • What devices can iCANgraph control?
    Currently, iCANgraph can control lighting, shutters, control panels and internal timers.
  • What happens if iCANgraph loses the network connection?
    If a network connection fails iCANgraph will warn the user. When the network becomes available again it will be picked up automaticaly, the current device status will gradually update and everything will be back to normal.
  • Does iCANgraph support timers?
    Yes. iCANgraph has its own configurable timers with multiple schedules. This makes unnecessary the presence of a physical timer device.
  • How many devices can iCANgraph support?
    iCANgraph has no limit on the number of devices that it can control, up to the limits imposed by the available memory in the system.
  • How many scenarios can a Control Panel have?
    iCANgraph does not impose any limit on the number of scenarios, thus the only limits are those of the physical Control Panel devices.
  • Can a Control Panel scenario be reconfigured?
    Yes. iCANgraph supports Control Panel scenario programming by allowing users to save the current state to a scenario with a single click.
  • Can iCANgraph be used with touch screens?
    Yes. iCANgraph has been designed for usage with any standard point and click device, including touchscreens.
  • Can iCANgraph be used remotely?
    Yes. iCANgraph can be used remotely if a remote connection software is installed in the operating system.
  • What protocols does iCANgraph support?
    Currently iCANgraph supports the CAN protocol through the serial port (RS232).
  • What ports does iCANgraph support?
    Currently iCANgraph supports serial RS232 ports. Note that USB serial ports are not supported.
  • Is it possible to load projects from other iLight software?
    No. Currently iCANgraph loads and saves projects in its own format.
  • What image formats does iCANgraph for the floor plan backgrounds?
    Currently iCANgraph can only load images in Windows Meta File (WMF) file format.

    • More information is available in the iCANgraph web site and in the Datasheet section.
