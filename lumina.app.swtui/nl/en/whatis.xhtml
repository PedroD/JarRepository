What is iCANgraph?
 

iCANgraph is a real-time monitoring and control application designed to simplify the command of iLight lighting systems.
This software extends installations of any size with automation, unlimited user-definable scenarios and power consumption reporting.

iCANgraph allows iLight device control and management through easy and fast mouse control graphical interface.
iCANgraph can control unlimited lamps, shutters, control panels and internal timers using a floor plan and area hierarchical arrangement.
The provided timers are configurable with multiple schedules and do not require the presence of any physical timer device.
The control panels scenarios can save the current device configuration with a single click. Control panel can also support unlimited scenario labels.
If a network connection fails, iCANgraph will warn the user and everything will be back to normal when the network becomes available again.

More information is available in the iCANgraph web site and in the Datasheet section.
