Key Features
 

iCANgraph key features are described below.

  • Control
    Device status is provided both through graphical and textual representations, allowing for a fast overview or a detailed view.
  • Automation
    Command execution can be automated through the usage of timer controls. This allows a system automation with a detail that has never been achieved before.
  • Power Consumption
    Power consumption is provided in real time. The consumption is grouped from single devices to entire building.
  • Reporting
    Power consumption and saving reporting are available on spreadsheet format.
  • Easy to Use
    The graphical interface interaction turns the command of any installation in a simple and easy task, no matter how wide it is. Navigation through floors, plans and areas enriches use experience.
  • Easy to Program
    Scenario reconfiguration allows fast light adjustments, specially suited to spaces that require dynamic light changes.
  • Real Time Information
    The real time device information provides rigorous floor, area and device control.

    • More information is available in the iCANgraph web site and in the Datasheet section.
